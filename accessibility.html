Accessibility Statement

Last updated: February 2025

Our Commitment

Zak Consult is committed to ensuring that our website is accessible to all people, including persons with disabilities. We strive to conform to the Israeli Standard SI 5568, which is based on the Web Content Accessibility Guidelines (WCAG) 2.1 at Level AA, as required by the Equal Rights for Persons with Disabilities Regulations (Service Accessibility Accommodations), 2013.

Accessibility Features

The following measures have been implemented on this website:

  • Keyboard Navigation: All content and interactive elements are fully accessible via keyboard. A visible focus indicator is provided for all interactive elements.
  • Skip Navigation: A "Skip to main content" link is provided at the top of each page to allow users to bypass repetitive navigation.
  • Semantic HTML: Proper semantic structure is used throughout, including landmarks (<nav>, <main>, <header>, <footer>), heading hierarchy, and ARIA labels where needed.
  • Color Contrast: Text and interactive elements meet the minimum contrast ratio of 4.5:1 for normal text and 3:1 for large text, as required by WCAG 2.1 AA.
  • Text Resizing: The website supports text resizing up to 200% without loss of content or functionality. An accessibility toolbar is provided for adjusting font size.
  • Alternative Text: All meaningful images include descriptive alternative text. Decorative images are marked appropriately.
  • Form Accessibility: All form fields have visible labels, required fields are clearly indicated, and validation errors are announced to assistive technology via ARIA live regions.
  • Accessibility Toolbar: A built-in accessibility toolbar allows users to increase or decrease text size, toggle high contrast mode, toggle grayscale, underline all links, and switch to a more readable font.
  • Reduced Motion: Animations and transitions are automatically disabled for users who have indicated a preference for reduced motion in their operating system settings.
  • Screen Reader Compatibility: The website has been designed to work with assistive technologies including NVDA and VoiceOver.

Standard Compliance

This website is designed to conform to:

  • Israeli Standard SI 5568 (based on WCAG 2.1)
  • Web Content Accessibility Guidelines (WCAG) 2.1, Level AA
  • The Equal Rights for Persons with Disabilities Law, 1998
  • The Equal Rights for Persons with Disabilities Regulations (Service Accessibility Accommodations), 2013

Browser & Assistive Technology Compatibility

This website is optimized for use with:

  • Google Chrome (latest version)
  • Mozilla Firefox (latest version)
  • Microsoft Edge (latest version)
  • Apple Safari (latest version)
  • Screen readers: NVDA, VoiceOver (macOS/iOS)

Known Limitations

While we strive for full accessibility, the following areas may have limitations:

  • The company logo image contains stylized text that may not be fully readable by all assistive technologies; descriptive alt text is provided as an alternative.
